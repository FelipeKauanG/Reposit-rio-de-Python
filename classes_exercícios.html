Criando uma classe para representar um retângulo

Crie uma classe Retangulo que representa um retângulo com altura e largura. A classe deve ter os seguintes métodos:

Método Descrição
__init__(self, largura, altura) Inicializa os atributos de largura e altura do retângulo.
area(self) Retorna a área do retângulo.
perimetro(self) Retorna o perímetro do retângulo.
diagonal(self) Retorna o comprimento da diagonal do retângulo.
escala(self, escala) Escala o retângulo por um fator especificado.

Adicione os métodos area, perimetro e diagonal à classe Retangulo. Esses métodos devem calcular e retornar a área, o perímetro e a diagonal do retângulo, respectivamente.

class Retangulo:
    def __init__(self, largura, altura):
        self.largura = largura
        self.altura = altura

    def area(self):
        return self.largura * self.altura

    def perimetro(self):
        return 2 * (self.largura + self.altura)

    def diagonal(self):
        return (self.largura ** 2 + self.altura ** 2) ** 0.5
